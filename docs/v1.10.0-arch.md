## 1. 架构概览

本期改动保持**纯前端单页应用**架构，不引入后端服务或数据库存储。所有状态变更通过内存中的 JavaScript 变量实现，页面刷新后状态重置（符合现有设计）。

```

│           浏览器 (Client)                │

│  └────┬────┘  └────┬────┘  └────┬────┘ │
│       └─────────────┴─────────────┘     │
```

## 2. 核心模块变更

### 2.1 模块：版本管理（versions.json）

**变更点**：新增 v1.10.0 版本条目，更新 `current` 字段。

**数据流**：
```
versions.json ──→ initApp() ──→ versionInfo DOM
```

### 2.2 模块：交易日识别与数据初始化

**新增函数**：`ensureTodayData()`

**职责**：
1. 检查 `todayText()` 是否在 `DAYS` 中
2. 如果不在，克隆 `latestDay()` 生成今日数据骨架
3. 更新 `currentDate` 到今天

**状态机**：
```
[页面加载] ──→ ensureTodayData() ──→ 判断 today 是否在 DAYS ──→
    ├─ 是 → 什么都不做
    └─ 否 → 克隆 latestDay → 更新 date/weekday → push 到 DAYS → currentDate = today
```

**克隆策略**（深拷贝避免引用污染）：
```javascript
const todayData = JSON.parse(JSON.stringify(latestDay()));
todayData.date = todayText();
todayData.weekday = getWeekdayText();
// price 保持为 latestDay 的收盘价（作为基准）
// prev 保持为 latestDay 的 price（昨收）
todayData.theme = '';
todayData.sectorName = '';
todayData.summary = '';
// ... 其他字段重置
```

### 2.3 模块：实时行情刷新

**变更函数**：`refreshLivePnL()`、`isTradingRefreshWindow()`、`startLiveRefresh()`

**时序图**：
```
initApp()
    │
    ▼
startLiveRefresh()
    │
    ├─ 立即执行 refreshLivePnL()
    │
    ▼
setInterval(10000ms)
    │
    ▼
isTradingRefreshWindow() ──→ 判断 9:30 ≤ now ≤ 15:15 ──→
    ├─ false → 跳过
    └─ true  → refreshLivePnL()
                    │
                    ▼
            loadSinaQuotes(codes)
                    │
                    ▼
            更新 todayData.price
                    │
                    ▼
            更新 INTRADAY_PNL（追加/更新当前时间点）
                    │
                    ▼
            render() + renderCharts()
```

**INTRADAY_PNL 动态构建算法**：
```javascript
function updateIntradayPnl(todayData, fundDaily) {
  const now = timeText(); // "HH:MM"
  const stockMv = calcStockMarketValue(todayData.price);
  const stockPrevMv = calcStockMarketValue(todayData.prev);
  const totalPnl = (stockMv - stockPrevMv) + fundDaily;
  
  const existingIdx = INTRADAY_PNL.findIndex(p => p.time === now);
  if (existingIdx >= 0) {
    INTRADAY_PNL[existingIdx].value = totalPnl;
  } else {
    INTRADAY_PNL.push({ time: now, value: totalPnl });
  }
}
```

**参数变更**：

| 开始时间 | 9:15 | 9:30 |

## 3. 数据模型

### 3.1 DAY 对象（新增字段说明）

当今日数据由系统自动生成时，以下字段有特殊含义：

| 字段 | 来源 | 说明 |
|------|------|------|
| `date` | todayText() | 今天日期 |
| `weekday` | 动态计算 | 今天星期几 |
| `price` | 实时 API | 初始为昨收，刷新后更新为实时价 |
| `prev` | latestDay.price | 昨收价，用于计算当日涨跌 |
| `theme` | 空字符串 | 新交易日无主题数据 |
| `sectorName` | 空字符串 | 新交易日无板块数据 |
| `summary` | 空字符串 | 新交易日无市场总结 |
| `idx` | latestDay.idx | 复制昨日指数（当日指数无实时）|

### 3.2 INTRADAY_PNL 动态构建

```typescript
interface IntradayPnlPoint {
  time: string;   // "HH:MM"
  value: number;  // 当日总盈亏（证券+基金）
}
```

**构建规则**：
- 9:30 首次刷新 → 添加第一个点
- 每 10 秒刷新 → 如果当前分钟已存在则更新，否则追加
- 15:15 后停止追加

## 4. 关键逻辑伪代码

### 4.1 ensureTodayData
```javascript
function ensureTodayData() {
  const today = todayText();
  const hasToday = DAYS.some(d => d.date === today);
  if (hasToday) return;
  
  const latest = latestDay();
  const todayData = JSON.parse(JSON.stringify(latest));
  todayData.date = today;
  todayData.weekday = getWeekdayText(new Date());
  todayData.prev = { ...latest.price }; // 昨收 = 昨日收盘价
  todayData.theme = '';
  todayData.sectorName = '';
  todayData.summary = '交易日实时数据，盘中自动更新';
  todayData.limitUp = 0;
  todayData.ud = '';
  todayData.volume = '';
  todayData.direction = '';
  todayData.earnEffect = '';
  todayData.sentiment = '';
  todayData.strength = '';
  
  DAYS.push(todayData);
  currentDate = today;
}
```

### 4.2 refreshLivePnL（修改后）
```javascript
async function refreshLivePnL() {
  if (!isTradingRefreshWindow()) return;
  
  const today = todayText();
  let todayData = DAYS.find(d => d.date === today);
  
  // 如果今天数据不存在（理论上 ensureTodayData 已处理），兜底创建
  if (!todayData) {
    ensureTodayData();
    todayData = DAYS.find(d => d.date === today);
  }
  
  try {
    const codes = Object.keys(HOLDINGS.shares);
    const quotes = await loadSinaQuotes(codes);
    
    codes.forEach(code => {
      const q = quotes[code];
      if (!q) return;
      todayData.price[code] = q.price;
      todayData.prev = todayData.prev || {};
      todayData.prev[code] = q.prevClose;
      if (STOCK_INFO[code]) STOCK_INFO[code].price = q.price.toFixed(3);
    });
    
    // 更新 INTRADAY_PNL
    const fundDaily = FUND_PORTFOLIO.reduce((s, f) => s + Number(f.dailyIncome || 0), 0);
    const stockCodes = Object.keys(HOLDINGS.shares);
    const stockMv = stockCodes.reduce((s, c) => s + HOLDINGS.shares[c] * todayData.price[c], 0);
    const prevMv = stockCodes.reduce((s, c) => s + HOLDINGS.shares[c] * (todayData.prev[c] || todayData.price[c]), 0);
    const totalPnl = (stockMv - prevMv) + fundDaily;
    
    const nowTime = timeText();
    const existingIdx = INTRADAY_PNL.findIndex(p => p.time === nowTime);
    if (existingIdx >= 0) {
      INTRADAY_PNL[existingIdx].value = totalPnl;
    } else {
      INTRADAY_PNL.push({ time: nowTime, value: totalPnl });
    }
    
    lastLiveUpdateText = timeText();
    if (currentDate === today) {
      render();
      renderCharts();
    }
  } catch (err) {
    lastLiveUpdateText = `刷新失败 ${timeText()}`;
  }
}
```

### 4.3 isTradingRefreshWindow（修改后）
```javascript
function isTradingRefreshWindow(now = new Date()) {
  const day = now.getDay();
  const mins = now.getHours() * 60 + now.getMinutes();
  return day >= 1 && day <= 5 && mins >= 9 * 60 + 30 && mins <= 15 * 60 + 15;
}
```

## 5. 兼容性考虑

1. **向后兼容**：`ensureTodayData` 仅在页面加载时执行一次，不影响已有数据日
2. **非交易日**：周末/节假日 `isTradingRefreshWindow` 返回 false，不创建 today 数据
3. **API 失败**：`loadSinaQuotes` 失败时，`INTRADAY_PNL` 保留已有数据，页面不崩溃
4. **缓存策略**：portfolio.json 仍带 `?v=APP_VERSION&t=Date.now()` 防缓存
