> 对应 PRD：v1.10.0-prd.md  
> 对应 Arch：v1.10.0-arch.md  
> 编写日期：2026-07-01

---

## 1. 设计原则

- **一致性**：新交易日的UI表现与历史交易日保持一致，仅在数据缺失处做降级提示
- **实时感**：通过状态标签、时间戳、动态曲线传达"实时刷新"感知
- **容错性**：API失败或数据缺失时，不显示报错，使用占位文案或空状态

---

## 2. 页面级UI变更

### 2.1 版本弹窗（R1）

**位置**：右上角 `!` 按钮点击弹窗

**变更点**：
- `v1.10.0` 条目显示为 **当前版本**（高亮边框）
- `deploy_url` 为 "待预览确认后部署" 时，显示灰色占位文字
- 历史版本保持原有样式不变

**样式**：
```css
.version-item.current {
  border-left: 3px solid var(--accent);
  background: rgba(68,138,255,0.08);
}
.version-deploy-url.pending {
  color: var(--text-dim);
  font-style: italic;
}
```

### 2.2 新交易日首页/持仓页（R3+R4）

**顶部日期栏**：
- 日期选择器默认选中 **今天**（2026-07-01 周三）
- 后一天按钮禁用（因为今天已是最新）

**持仓总览区域**：
- `theme` 为空时，headerSub 显示为：`2026-07-01 周三 · — · —`
- 不显示 "震荡偏强" 等强度标签（因为无数据）
- "当日总收益" 使用实时计算值，颜色随正负变化

**数据更新时间标签**：
- 显示：`数据更新时间：实时计算｜最后刷新 10:05:30`

### 2.3 实时状态标签（R4+R6）

**位置**："趋势图表" 标题右侧

**状态样式**：

| 状态 | 文案 | 样式 |
|------|------|------|
| 交易时段刷新中 | `实时刷新中｜10:05:30` | 绿色脉冲圆点 + `#00c853` 文字 |
| 非交易时段 | `非交易刷新时段，数据固定｜最后更新 10:05:30` | 灰色 `#8b8fa3` 文字 |
| 刷新失败 | `刷新失败 10:05:30` | 红色 `#ff1744` 文字 |

**脉冲动画**：
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
.live-status::before {
  content: '';
  display: inline-block;
  width: 6px; height: 6px;
  border-radius: 50%;
  background: #00c853;
  margin-right: 4px;
  animation: pulse 1.5s infinite;
}
```

### 2.4 当日盈亏曲线（R5）

**有数据时**：
- 正常显示折线、圆点、零线、时间标签
- 最后一个点高亮（大圆点 + 数值标注）

**无数据时（新交易日开盘前）**：
- 不显示 "暂无数据" 空白，而是显示一个居中的提示：
  - 文案：`交易时段开始后自动刷新`
  - 样式：14px, `var(--text-dim)`, 垂直居中

**逐步构建中**：
- 1-2 个点时正常连线
- 随着点增加，曲线自然延伸
- 零线始终可见

### 2.5 当日盈亏归因（R7）

**有板块数据时**：原有样式不变

**无板块数据时（新交易日）**：
- 不归零显示空图，而是显示友好提示：
```html
<div style="text-align:center;padding:40px 20px;color:var(--text-dim);font-size:.85rem">
  <div style="margin-bottom:8px;font-size:1.5rem">📊</div>
  <div>今日暂无板块归因数据</div>
  <div style="font-size:.75rem;margin-top:4px;opacity:.7">盘中数据更新后将自动展示</div>
</div>
```

### 2.6 日历视图（R3）

**新交易日显示**：
- 今天的日期格用 **虚线边框** 标识（区别于实线边框的历史日期）
- 盈亏数值显示实时计算值
- 无盈亏时显示 "—"

## 3. 交互设计

### 3.1 定时刷新交互

- 用户无感知：后台每10秒刷新，不弹窗、不阻塞
- 数据变化时：页面平滑更新（DOM diff 由 render() 处理）
- 网络异常时：保留上一次数据，状态标签变红提示

### 3.2 Tab切换（兼容性）

- 切换到"全部"/"证券"/"基金"时，当日盈亏曲线保持显示（不受 summaryTab 影响）
- 归因图根据 Tab 显示/隐藏（基金 Tab 显示 "基金不展示个股归因"）

## 4. 响应式与主题

### 暗色主题
- 实时状态绿色 `#00c853`，红色 `#ff1744`
- 空状态提示使用 `var(--text-dim)`（`#8b8fa3`）

### 亮色主题
- 实时状态绿色调整为 `#00a040`
- 空状态背景保持 `#f5f7fb`

### 移动端
- 当日盈亏曲线高度保持 200px，不压缩
- 时间标签字号在 375px 宽屏下缩小至 7px
- 版本弹窗全屏显示
