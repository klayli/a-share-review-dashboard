## 1. 测试策略

### 1.1 测试目标
验证版本说明管理、新交易日实时收益、定时刷新三个核心功能的正确性，确保部署流程合规。

### 1.2 测试范围
- 功能测试：版本弹窗、数据初始化、实时计算、定时任务、空状态
- 兼容性测试：历史日期/今日切换、Tab联动、明暗主题
- 异常测试：API失败、非交易日、周末、网络中断
